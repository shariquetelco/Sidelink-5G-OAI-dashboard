## Component Details

### Unit Alpha (SyncRef UE)
- **Role:** Synchronization Reference
- **Function:** Broadcasts PSBCH for timing/frequency sync
- **Configuration:**
  - Carrier: 2.6 GHz
  - Bandwidth: 106 RBs
  - MCS: 9
  - TX Power: 23 dBm

### Unit Bravo (Nearby UE)
- **Role:** Synchronized Receiver
- **Function:** Detects S-PSS/S-SSS, decodes PSBCH, maintains sync
- **Configuration:**
  - Carrier: 2.6 GHz
  - Bandwidth: 106 RBs
  - MCS: 9
  - RX Gain: Auto

### Dashboard Components
1. **Backend (Flask):** Python REST API
2. **Parser:** Log file analyzer with regex
3. **Frontend:** Tactical-themed HTML/CSS/JS
4. **Real-time Engine:** JavaScript fetch every 2s
